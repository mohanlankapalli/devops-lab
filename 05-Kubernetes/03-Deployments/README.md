# Kubernetes Deployment

## Objective

Learn how to deploy an application using a Deployment.

## What is a Deployment?

A Deployment manages Pods and ReplicaSets. It ensures the desired number of replicas are running and supports rolling updates and rollbacks.

## Why use Deployments?

- High availability
- Self-healing
- Rolling updates
- Easy scaling
